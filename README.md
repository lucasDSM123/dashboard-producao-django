Dashboard de Produção - Django
Sistema desenvolvido para gestão e análise de materiais de produção, focado em automação e visualização de dados financeiros por obra.

🚀 Funcionalidades
Dashboard Interativo: Visualização de investimentos totais e por obra através de gráficos dinâmicos.

Filtros Inteligentes: Consulta de dados por períodos específicos.

Exportação: Geração de relatórios em Excel utilizando a biblioteca Pandas.

Gestão de Materiais: Cadastro completo com precificação automática via tabela interna.

🛠 Tecnologias
Python / Django

JavaScript (Chart.js)

Pandas / Openpyxl

CSS / HTML5
